# Claude Code Skills for Marketing & Communications Directory

A curated directory of real, installable Claude Code skills built by the community — filtered for marketing and communications work. No developer background needed to use them.

**→ [Open the Directory](https://monitalan.github.io/claude-code-skills-directory/)**

---

## What is this?

Claude Code skills (also called agent skills) are installable instruction packages that give AI coding agents specialized expertise. This directory cuts through the noise and surfaces only the skills relevant to **marketers, communications professionals, and brand strategists**.

Every entry links to a real GitHub repo with a working install command.

---

## Categories

- **Content & Brand** — content creation, brand voice, guidelines
- **Analytics & CRO** — campaign analytics, conversion optimization, social analysis
- **Strategy & SEO** — GTM, positioning, app store optimization
- **Collections** — curated bundles and awesome lists

---

## How to use a skill

1. Install [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
2. Find a skill in the directory
3. Copy the install command
4. Run it in your terminal
5. Ask Claude to use the skill in plain English — no coding required

---

## Submit a skill

Found a skill that belongs here? Open a Pull Request with:

- Skill name and GitHub repo URL
- 1–2 sentence description
- Relevant categories
- Install command

**Requirements:** must be publicly available on GitHub, have a working `SKILL.md`, and be relevant to marketing or communications work.
